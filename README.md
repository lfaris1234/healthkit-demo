# HealthKit Demo App

A demo web application built with **Next.js**, **TypeScript**, and **TailwindCSS**, inspired by the attached mockup.

## Tech Stack
- Next.js 14
- TailwindCSS
- TypeScript
- Zustand for state management
- Axios for API calls
- Optional: Prisma + SQLite/PostgreSQL

## Setup

```bash
npm install
npm run dev
